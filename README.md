# PetShopServer

## 개발모드로 시작하기
1. git clone https://github.com/donggyushin/PetShopServer.git
2. yarn install
3. .env 파일 추가하기
4. yarn dev

## 배포하기
1. git clone https://github.com/donggyushin/PetShopServer.git
2. yarn install
3. .env 파일 추가하기
4. yarn build
5. yarn start:production (테스트 서버 실행은 yarn start:test)
